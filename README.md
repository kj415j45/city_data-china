# 中国城市区划数据（三级）

按省、市、区顺序分类给出 `json` 格式数据，方便选择地址。

## 构建数据

**⚠ 警告：该脚本为爬虫，运行脚本的风险与后果请自行承担。**

运行 php 脚本后即可在 `dist/` 目录下找到数据。

```sh
php main.php
```
